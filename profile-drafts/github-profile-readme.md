# Roy Yali Samaniego

**Geocomputing & GeoAI Researcher · Earth Observation · GIS Software · Applied Innovation**

I work at the intersection of geocomputing, GeoAI, Earth Observation and GIS software. My work connects applied research, open-source tools and spatial data systems for environmental monitoring, remote sensing, WebGIS and 3D geospatial reconstruction.

I am currently pursuing a PhD in Geotechnologies and building a research agenda around reproducible geospatial AI, cloud-native spatial data workflows and applied innovation for territorial and environmental challenges.

## Focus areas

- GeoAI and semantic segmentation for satellite imagery
- Earth Observation pipelines with Python, R, GEE, STAC and cloud platforms
- GIS software, WebGIS, APIs and spatial databases
- Environmental monitoring, protected areas, forest and risk analysis
- 3D reconstruction with SfM, NeRF and Gaussian Splatting
- Teaching geospatial programming and reproducible research

## Selected work

- **CloudSEN12** — global dataset for cloud and shadow semantic understanding in Sentinel-2.
- **rgee** — R interface for Google Earth Engine.
- **WebGIS / spatial data systems** — PostGIS, GeoServer, FastAPI, NodeJS, React/Angular.
- **GeoAI monitoring workflows** — semantic segmentation, change detection and satellite data harmonization.
- **Geospatial teaching** — programming for spatial analysis with Python and R.

## Long-term vision

I want to help build a strong research group in Peru focused on geocomputing, GeoAI, Earth Observation and open geospatial technologies.
